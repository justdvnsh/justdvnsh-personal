# Personal Website

This is my personal website

## Getting Started
```
* clone the repo
* make the mongo db connection 
* sign up as you the user to get the admin funcitonality.
* fire up the browser
* enjoy
```

### Prerequisites

What things you need to install the software and how to install them

```
* NodeJS
* MONGO DB
```

## Built With
```
* Javascript
* NodeJS
* MONGODB
```
